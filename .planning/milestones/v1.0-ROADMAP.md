## Milestone Summary

**Key Decisions:**
- Near-full replica fidelity for authentic macOS experience
- All 5 apps in v1 for complete app layer polish
- YOLO workflow mode for faster iteration
- execCommand for rich text formatting (no viable replacement)
- VirtualFileSystem shared between Terminal and Finder
- localStorage for Notes and filesystem persistence

**Issues Resolved:**
- Fixed dock click-to-open/focus functionality
- Fixed format button state tracking with selectionchange
- Fixed focus preservation on toolbar buttons
- Fixed Terminal VirtualFileSystem integration
- Implemented soft delete for Notes

**Technical Debt:**
- Initial Finder window uses AppType::Generic instead of AppType::Finder
- Calculator/TextEdit/Terminal state not persisted on window close
- Dock has unimplemented apps (Safari, Messages, etc.)
- User-reported: dock icon sizes, finder padding, calculator clipping
